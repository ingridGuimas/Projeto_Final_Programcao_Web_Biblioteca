# Sistema de Biblioteca

Este projeto é um sistema simples de gerenciamento de biblioteca desenvolvido em **PHP** com **MySQL**, utilizando **Bootstrap** para estilização.  
Ele permite o cadastro, listagem, edição e exclusão de **Usuários**, **Livros** e **Reservas**, seguindo o padrão CRUD.

---

## Sobre o Projeto

O sistema permite gerenciar:

- **Usuários**
- **Livros**
- **Reservas**

Cada módulo possui operações de:

- **Cadastrar**
- **Listar**
- **Editar**
- **Excluir**

## Como Executar o Projeto

1. Coloque o projeto em uma pasta dentro do **XAMPP** ou **WAMP**  
   Exemplo: htdocs/biblioteca/
   
3. Importe o banco de dados que está na pasta **database/**.

4. Inicie os serviços **Apache** e **MySQL**.

5. Acesse o sistema pelo navegador: http://localhost/Projeto_final_programcao_web/
